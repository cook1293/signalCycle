# signalCycle

## 概要
信号交差点における信号サイクルアニメーションを再現したJavaアプリケーション。

## 開発環境・言語
Eclipse 3.5 (Galileo)，Java

## ファイル構成
- SignalNormal.java：通常の信号交差点の信号サイクル
- SignalButton.java：押しボタン式信号交差点の信号サイクル
